## Image Guidelines

- **Format**: JPG for photos
- **Size**: Optimize to be under 500KB when possible
- **Dimensions**: Minimum 800x600px, ideally 1200x800px for program cards
- **Content**: Should show the trust's activities, beneficiaries, or facilities
