## Image Guidelines

- **Format**: JPG for scanned certificates
- **Size**: Optimize to be under 500KB when possible
- **Dimensions**: Maintain aspect ratio of original certificates
- **Quality**: High resolution scans so text is readable when zoomed
- **Content**: Official certificates, awards, and recognitions only
