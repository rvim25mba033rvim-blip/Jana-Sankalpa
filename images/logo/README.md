# Logo Directory

This directory contains the logo files for Jana Sankalpa Seva Charitable Trust (R).

## Logo Placement

The logo is displayed in the header section at the beginning, before the organization name "Jana Sankalpa Seva Charitable Trust (R)".

## Required Files

Place your logo file in this directory with one of the following names:
- `logo.png` (recommended)
- `logo.jpg`
- `logo.svg`

## Specifications

- **Recommended size**: 44px × 44px (to match the current header height)
- **Format**: PNG with transparent background (preferred) or JPG/SVG
- **Aspect ratio**: Square or close to square for best results
- **Color**: Should work well with the website's color scheme (red and white theme)

## Usage

The logo is referenced in `index.html` within the header navigation section and styled using CSS in `css/style.css`.
