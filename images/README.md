# Images Folder

This folder contains all images used in the Jana Sankalpa Seva Charitable Trust website.

## Folder Structure

```
images/
├── README.md           # This file
├── hero/              # Hero section images and videos
├── about/             # About section images
├── team/              # Team member photos
├── awards/            # Award certificates and recognition images
└── activities/        # Activity and event photos
```

## Image Files to Add

### Hero Section (`hero/`)
- `hero_video.mp4` - Background video for hero section
- `trust_banner.jpg` - Fallback banner image for hero video

### About Section (`about/`)
- `building.jpg` - Main ashrama building photo
- `activities_banner.jpg` - Community activities photo

### Team Section (`team/`)
- `jyothi.jpg` - Smt. Jyothi S. (Chairperson)
- `narasimhaiah.jpg` - Dr. B. Narasimhaiah (President)
- `geetha.jpg` - Shri Devarakumar Geetha (Secretary)
- `naveen.jpg` - Shri Naveen Vajrapaksha (Vice President)

### Awards Section (`awards/`)
- `award_certificate.jpg` - Praja Seva Kannadada Ratna certificate
- `govt_award.jpg` - Karnataka Government recognition certificate

## Adding New Images

1. Place images in the appropriate subfolder based on their purpose
2. Use descriptive filenames (e.g., `team-member-name.jpg`)
3. Optimize images for web (recommended: JPEG for photos, PNG for graphics)
4. Update the HTML file to reference the new image paths

## Image Guidelines

- **Format**: JPG for photos, PNG for graphics with transparency
- **Size**: Optimize images to be under 500KB when possible
- **Dimensions**: Hero images should be at least 1920x1080px
- **Naming**: Use lowercase with hyphens (e.g., `building-front.jpg`)
