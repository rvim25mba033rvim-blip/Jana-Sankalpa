## How to Add Photos

1. Get professional headshots or formal photos of each team member
2. Name the files exactly as listed above (lowercase, no spaces)
3. Place them in this folder (`images/team/`)
4. The website will automatically display them in the Team section

## Image Guidelines

- **Format**: JPG for photos
- **Size**: Optimize to be under 300KB per photo
- **Dimensions**: Square format recommended (e.g., 400x400px or 600x600px)
- **Style**: Professional headshots with clean background
- **Quality**: High resolution, well-lit, clear facial features
